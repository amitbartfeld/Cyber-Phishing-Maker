About this project

This project was created by Amit Bartfeld and Orel Pashayev. To rise phishing awareness.

We made this website to show people how easy it is to create a phishing site, from any website they know on the internet.

This project was made using Python, HTML, JS, and CSS.
It's our final project for cyber security course in college.

We hope this project will help spread awareness for phishing and will help people to protect themselves.

Thank you for using our project.
